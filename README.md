## Rolls By The Pound

### Project Description  

Rolls By The Pound is a small-business bakery located in Newark, OH. This web application serves as an online odering service for their bakery. The app allows users to create a profile, browse the menu, add items to their cart, customize items, and checkout. The users have access to past orders that allows them to reorder from that dashboard.

### Tech Stack

Front and Backend: Next.js  
Database: PostgreSQL via Supabase
